# Long Term Investing Vault

Ruby Osborne's personal investing system. Long-term US equities, NZ-based investor, $250/week DCA via Sharesies.

The Monday brief runs as a Claude Code routine on Anthropic's cloud at 17:00 NZT. Mid-week insights flow through the Claude mobile app and append to ops/inbox.md. All trades are placed manually by Ruby.

See `self/` for methodology and `Investing Knowledge Base/` for the knowledge graph.
