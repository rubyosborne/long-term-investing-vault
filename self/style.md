# Writing style

- No em dashes. Use full stops or commas.
- No consulting buzzwords (synergy, leverage as verb, ecosystem, mission-critical, etc.)
- Plain NZ English. "Realise" not "realize". "Organisation" not "organization".
- Punchy, direct sentences. Short paragraphs.
- Default deliverable format: markdown. Use .docx only when explicitly requested for the brief.
- Never delete a file without explicit Ruby approval. Move to a Processed/ subfolder instead.
- When presenting picks, lead with the conviction rating and bull thesis in one sentence. Then bear case. Then numbers.
